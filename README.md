# Kill that guy!

A simple browser game. You need to kill a goblin.

### Features:
- 1 second to kill
- 5 kills to win
- 5 misses to lose

Do your best!

Try it on [GitHub Pages](https://solarlime.github.io/kill-that-guy/)!
